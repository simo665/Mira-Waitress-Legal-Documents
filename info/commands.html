🛠️ Server Management

🚪 Join/Leave Channels

/joins channel

Set welcome channel

/leaves channel

Set farewell channel

/joins embed

Customize welcome message

Welcome Message Example

📜 Logging

/logging add-channel

Add logging channel

/logging remove-channel

Remove logging channel

Server Logging Example

🤖 Auto Reactions

/autoreact add

Add reaction triggers

Note: Separate multiple words and emojis with commas
Example:
reactions: ❤️, 👋
words: hey, welcome, Hi
/autoreact remove

Remove reaction triggers

Auto Reactions Example

💎 Booster Features

/booster channel

Set booster channel

Note: Server must have at least one boost. Use /boosters role first.
Server Boost Board

🧵 Auto Threading

/thread add

Enable auto-threading

Optional: Add media-only parameter
/thread remove

Disable auto-threading

Auto Threading Example

🔨 Moderation

/warn

Warn a user

Can include mute duration
/mute

Mute a user

/kick

Kick a user

/ban

Ban a user

/unmute

Unmute a user

/unban

Unban a user

/purge

Delete up to 1,000 messages

/appeals_link

Set appeal link for punished users

/say

Send message as bot

Unauthorized mentions are automatically removed
/stick set

Stick a message

/stick remove

Remove stuck message

❕ Prefix Commands

m!channelslist

List all server channels

Purge messages

Stick a message

Remove stuck message

💡 Important Notes

  • Multiple configurations are supported simultaneously
  • New features and commands are added regularly
  • Slash commands (/) are the primary interaction method
  • Use commas to separate multiple words/emojis in commands
